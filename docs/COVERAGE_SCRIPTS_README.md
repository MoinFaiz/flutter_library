# Coverage Report Scripts

This repository contains two simple scripts to read and analyze LCOV coverage files generated by Flutter tests.

## Scripts Available

### 1. Dart Script (`coverage_report.dart`)
A Dart script that provides detailed coverage analysis with emojis and color-coded output.

#### Usage:
```bash
# Run with default lcov file (coverage/lcov.info)
dart coverage_report.dart

# Run with custom lcov file path
dart coverage_report.dart path/to/your/lcov.info
```

### 2. PowerShell Script (`coverage_report.ps1`)
A PowerShell script for Windows users that provides similar functionality without emojis for better compatibility.

#### Usage:
```powershell
# Run with default lcov file (coverage\lcov.info)
powershell -ExecutionPolicy Bypass -File coverage_report.ps1

# Run with custom lcov file path
powershell -ExecutionPolicy Bypass -File coverage_report.ps1 -LcovPath "path\to\your\lcov.info"
```

## Features

Both scripts provide:

- **Overall Coverage Summary**: Total files, lines, and coverage percentage
- **Per-File Coverage**: Detailed breakdown for each file
- **Coverage Distribution**: Files grouped by coverage ranges
- **Lowest Coverage Files**: Identifies files that need attention
- **Color-Coded Output**: Visual indicators for coverage quality
- **Sorted Results**: Files sorted by coverage percentage (lowest first)

## Coverage Quality Indicators

- **🟢 EXCELLENT (90-100%)**: Very good test coverage
- **🟡 GOOD (80-89%)**: Good test coverage
- **🟠 FAIR (70-79%)**: Acceptable but could be improved
- **🔴 POOR (0-69%)**: Needs attention

## Requirements

- **Dart Script**: Dart SDK installed
- **PowerShell Script**: Windows PowerShell 5.1+ or PowerShell Core

## LCOV File Format

These scripts parse standard LCOV files with the following structure:
- `SF:` - Source file path
- `DA:` - Line execution data
- `LF:` - Lines found (total)
- `LH:` - Lines hit (covered)
- `end_of_record` - End of file record

## Integration with Flutter

To generate coverage data in Flutter:

```bash
# Run tests with coverage
flutter test --coverage

# Generate LCOV report
genhtml coverage/lcov.info -o coverage/html

# Run our coverage analysis
dart coverage_report.dart
```

## Customization

Both scripts can be easily modified to:
- Change coverage thresholds
- Add more detailed analysis
- Export results to different formats (JSON, CSV, etc.)
- Filter files by path patterns
- Add branch coverage analysis (if available in LCOV data)
